До - функция первичного заполнения договора перевозки без участия оператора.

После 
Из-за мутирования состояния объекта, не понятна структура и предназначение основной функции,
явно нужен новый тип выходных данных для функции расчета - первичный договор перевозки. 

Размышлял следующим образом:
- Выделил новый выходной тип данных с инкапсуляцией логики внутри;
- Показал с помощью конструкторов, как строится выходной тип данных;
- Явно выделил метод для связи с основной сущностью - договором перевозки;
