До - класс для установки баннера для доп. вопросов.
есть дополнительная проверка - для проверки были ли такой баннер, чтобы его деактивировать соот-во.
При этом при деактивации шлется пуш чтобы обновить баннер, а при активации нет.

После - по сути изменил ментальную модель баннеров, пуши шлются всегда, это даже удобнее, соот-во никакие условные проверки, просто не нужны.
