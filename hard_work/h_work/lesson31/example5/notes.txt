До - налепили лишних абстракций оберток для прокидывания метаданных (такой сервис не один), чтобы ключ был инкапсулирован внутри.

После - типизируем ключи метаданных + и инкапсулируем текст ключа внутри.
Для того чтобы по Generic сущности достать ключ, можно использовать атрибут на каждый кастомный ключ.
(Либо через рефлексию брать статик поле)
В c#, к сожалению нельзя сделать Interface с static required полем, чтобы можно использовать в Generic-ах.
